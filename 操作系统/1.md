## 什么是操作系统
* 用户角度
  * 操作系统是一个控制软件
  * 为应用程序提供服务
    * IO服务
    * 声卡
    * 网卡
* 对他内部管理的对象
  * 操作系统时资源管理器

### 操作系统层次结构
* 硬件之上
* 应用程序之下

### 操作系统在软件的体系结构
* 应用软件
* 系统软件位于应用软件之下，为应用软件提供服务支撑
``` mermaid
graph TD;
    A[Software] --> B[Application]
    A --> C[System]
    C --> D[Utility]
    C --> E[OS]
    E --> Shell
    E --> Kernel
    Shell --> U["User 操作系统向外暴露的接口，提供的一种可见的服务"]
```

### 需要关注哪些
* 硬件资源
  * CPU
    * CPU调度
    * 进程线程管理
  * 内存
    * 物理内存的管理
    * 虚拟内存管理
      * 给上层应用提供尽可能大的内存，供其使用
  * 磁盘
    * 文件系统管理
* 中断处理与设备驱动

### OS Kernel的特征
* 并发
* 共享
* 虚拟
* 异步
